# ToDo List App

A simple and responsive ToDo List application built using **HTML**, **CSS**, and **JavaScript**.

This project helps users manage daily tasks by allowing them to add, edit, delete, and save tasks directly in the browser using **localStorage**.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Save tasks in browser with localStorage
- Display current date
- Responsive design for smaller screens

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
